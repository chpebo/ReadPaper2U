设置

所有设置仅保存在你的浏览器本地。
API
示例：https://api.deepseek.com、https://api.openai.com、https://generativelanguage.googleapis.com/v1beta/openai
例如 deepseek-chat、gpt-4o-mini、gemini-2.0-flash
默认按模型名自动识别（kimi-k2.6 ≈ 256K，moonshot-v1-128k ≈ 128K，deepseek-chat ≈ 64K…）。如果你用的是中转 / 自定义代理且实际窗口与模型名不符，在这里手动填 token 数（例：200000）。
仅保存在本地 localStorage，只会发往你配置的接口，不会上传到其他地方。
界面 / 体验
同时影响界面文案与系统提示词的输出语言。
控制脚本场景数量。改变这一项会破坏旧论文的提示词缓存。
视觉模型示例：base URL https://api.moonshot.ai / 模型 kimi-k2.6，或任何支持 OpenAI 多模态消息格式的接口。
